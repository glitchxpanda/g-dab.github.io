如果WSL里也装了VScode，那可以改环境变量, 或是卸载WSL里的VScode

Compiler 和 debugger

WSL 没有默认安装编译器，所以我们需要自己安装g++和gdb来compile和debug

install the GNU compiler tools （含g++） and the GDB debugger
